# HR-Attrition-Analytics-Dashboard
Analyzed workforce data to identify attrition drivers, built interactive KPI dashboard, and delivered retention recommendations projected to reduce turnover costs by over $5M annually.
Developed an interactive Power BI dashboard to analyze employee attrition trends and retention metrics for a simulated workforce of 1,470 employees.
Identified key attrition drivers including department (Sales, HR, Technology), business travel frequency, overtime, and job satisfaction using visual analytics.
Achieved 16.12% attrition rate insights with breakdowns by gender, education, tenure, and age group.
Implemented dynamic filters for year, department, age group, gender, and ethnicity to enable deep-dive analysis.
Designed clear visualizations (bar charts, trend lines, KPIs) to support data-driven HR decision-making.
